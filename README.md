# paintJS
- 일시 : 2021.05.19
- 인원 : 이혜성
- 내용 : Making an Paint Apllication with Vanila Javascript
- 구현 사항 : 
  - 이미지 드로잉
  - 라이 색상 변경
  - 캔버스 색상 변경
  - 변경 된 캔버스 색상에 여러 라인 색상 채색
  - 채색한 캔버스의 이미지를 저장버튼으로 눌러 이미지 파일 확장자에 따라 저장
  - 캔버스 내에서 마우스 우클릭 블락
